# Director Desktop Application

### Pre-requisits
- Rust

### Installation

```bash 
bun install
```

## Developpment

This will build the backend and start a development app:

```bash
bun run dev
```

Note: to open the inspector, use the `Command + Option + I` keyboard shortcut

## Build

This will build the backend and create a DMG.

```bash
bun run build
```
